# The Orchestrator's Manifesto: Mastering the Tripartite Workflow

> "The developer is no longer a manual typist, but a systems orchestrator overseeing digital engineering teams."

## I. The Tripartite Hybrid Architecture

The modern workflow is partitioned into three distinct operational nodes to bypass context degradation and rate limits:

1.  **The Orchestrator (Google Antigravity)**:
    - **Model**: Gemini 3.1 Pro.
    - **Role**: High-level architectural planning, database schema design, and dependency research.
    - **Mode**: Planning Mode (Deep Reasoning).
    - **Artifact**: Technical Specifications & Blueprints.

2.  **The Executor (Claude Code)**:
    - **Model**: Claude Sonnet 4.6 / Opus 4.6.
    - **Role**: Deterministic execution of specifications into multi-file codebase implementation.
    - **Mode**: Terminal-Native Precision.
    - **Artifact**: Functional, Synatactically Perfect Code.

3.  **The Validator (Testsprite / Automated CI)**:
    - **Role**: Continuous validation of the execution against the original architectural constraints.
    - **Feedback**: Feeds error logs back to the Orchestrator for self-correction.

## II. The Skill Pattern (Agentic Wisdom)

Behavioral logic must be decoupled from code. We use the `SKILL.md` format to define "Agentic Wisdom":

- **Frontmatter**: Metadata defining name, description, and activation triggers.
- **Instincts**: Rules and Hooks that steer agent behavior in real-time.
- **Compaction**: Strategic memory management to maintain long-term task state.

## III. Economic & Operational Optimization

- **Zero-Capital Inception**: Leveraging Antigravity's free tier for token-heavy research.
- **Deterministic Handoff**: Reducing Claude's token usage by providing unambiguous, pre-validated blueprints.
- **MCP Integration**: Using the Model Context Protocol to abstract infrastructure (Supabase, Vercel, Figma) into natural language commands.

## IV. The Evolution of the Developer

The developer's value is now measured by:
- **Loop Design**: Efficiency of the multi-agent feedback cycles.
- **Instructional Precision**: Quality of the behavioral rules in the `.agent` directory.
- **System Stewardship**: Auditing security and integration of the autonomous swarm.
